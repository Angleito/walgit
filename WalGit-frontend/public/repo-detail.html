WalGit Client

Client-side implementation and CLI tools for interacting with WalGit repositories on the Sui blockchain.

Features

  • Full Git-compatible command line interface
  • Blockchain transaction management with retry and batching
  • Multi-tier storage optimization for different code components
  • Credential management with secure key storage
  • Advanced merge strategies with conflict resolution

Installation

npm install -g walgit-client

Usage

walgit init
walgit add .
walgit commit -m "Initial commit"
walgit push

See the documentation for complete usage instructions.
